# Sudoku Premium

Mobile-first Sudoku app built with React + TypeScript + Vite.

## Stack
- React
- TypeScript (strict)
- Zustand
- Tailwind CSS
- Framer Motion
- LocalStorage + IndexedDB
- Capacitor Android

## Run
```bash
npm install
npm run dev
```

## Build Web
```bash
npm run build
```

## Build Android APK (Cloud)
Use GitHub Actions workflow: `.github/workflows/android-apk.yml`
